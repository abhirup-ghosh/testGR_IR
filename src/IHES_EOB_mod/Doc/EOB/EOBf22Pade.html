EOBf22Pade

PURPOSE

EOBf22Pade Computes the f22(x,nu) correction factor.

SYNOPSIS

function f22 = EOBf22Pade(x,nu)

DESCRIPTION 

EOBf22Pade Computes the f22(x,nu) correction factor.
   f_22(x,nu) is computed up to 5PN, although the nu-dependent terms are
   included only up to 3PN 

   f22 = EOBf22Pade(x,nu)
